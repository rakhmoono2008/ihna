# 🤖 Бот обращений граждан Узбекистана

Телеграм бот для приёма обращений от граждан с выбором языка, региона и возможностью отправки медиафайлов.

---

## 📋 Функционал

- 🌐 Выбор языка (Русский / O'zbek)
- 📞 Выбор: поделиться номером или остаться анонимным
- 🗺 Выбор региона (все 14 областей + Каракалпакстан + Ташкент)
- 📝 Отправка текстового сообщения + фото + видео + документы
- 📨 Пересылка сообщений администратору и/или в группу

## 🚀 Деплой на Railway

### Шаг 1 — Загрузите код на GitHub

```bash
git init
git add .
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/ВАШ_ЛОГИН/uzbekistan-bot.git
git push -u origin main
```

### Шаг 2 — Создайте проект на Railway

1. Зайдите на [railway.app](https://railway.app)
2. Нажмите **"New Project"**
3. Выберите **"Deploy from GitHub repo"**
4. Выберите ваш репозиторий `uzbekistan-bot`

### Шаг 3 — Добавьте переменные окружения

В панели Railway → вкладка **Variables** — добавьте:

| Переменная | Значение | Описание |
|------------|----------|----------|
| `BOT_TOKEN` | `8239199632:AAGF3i4Qsswck4WEhaeLqHNvGBgE1bb2uVw` | Токен вашего бота |
| `ADMIN_ID` | `123456789` | Ваш Telegram ID (узнать: @userinfobot) |
| `GROUP_ID` | `-1001234567890` | ID группы/канала (см. ниже) |

### Шаг 4 — Узнайте ваш Telegram ID

1. Напишите боту **@userinfobot** в Telegram
2. Он пришлёт ваш числовой ID — вставьте его в `ADMIN_ID`

### Шаг 5 — Узнайте ID группы

**Способ 1 (через бота):**
1. Добавьте вашего бота в группу
2. Дайте боту права администратора
3. Напишите в группе `/start`
4. Зайдите на: `https://api.telegram.org/bot<ВАШ_ТОКЕН>/getUpdates`
5. Найдите `"chat":{"id": ...}` — это и есть ID группы (обычно начинается с `-100`)

**Способ 2:**
1. Перешлите любое сообщение из группы боту **@getidsbot**

### Шаг 6 — Деплой

После добавления переменных Railway автоматически запустит бота.
Проверьте вкладку **Deployments** — должен быть статус ✅ **Active**.

## 🔧 Локальный запуск (для тестирования)

```bash
# Установить зависимости
pip install -r requirements.txt

# Создать .env файл
echo "BOT_TOKEN=8239199632:AAGF3i4Qsswck4WEhaeLqHNvGBgE1bb2uVw" > .env
echo "ADMIN_ID=ВАШ_ID" >> .env
echo "GROUP_ID=ID_ГРУППЫ" >> .env

# Запустить
python bot.py
```

## ⚠️ Важно

- Бот **не хранит** данные пользователей постоянно — только в рамках одной сессии
- Если `GROUP_ID = 0` — сообщения будут приходить только администратору
- Если `ADMIN_ID = 0` — только в группу
- Бот принимает: текст, фото, видео, документы
